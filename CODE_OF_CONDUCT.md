# Code of Conduct

Message, Unpacked. is a public-interest education project. Participation should
make students, teachers, contributors, and people affected by fraud safer and
more respected.

## Expected behaviour

- Discuss ideas and evidence without attacking the person presenting them.
- Use language that is respectful of victims and avoids blame, shame, or
  stereotypes about age, education, disability, or technical skill.
- Give review feedback that is specific, actionable, and proportionate.
- Protect personal and sensitive information. Use reserved examples when
  reporting a problem.
- Disclose factual, licensing, safety, or accessibility uncertainty instead of
  hiding it to speed up publication.
- Respect educators' and maintainers' decisions to pause a sensitive case for
  further review.

## Unacceptable behaviour

Harassment, threats, sexualized conduct, discriminatory language, deliberate
misinformation, victim-blaming, publishing private information, sharing active
malicious links, or using project spaces to target students are not accepted.

## Reporting and enforcement

For ordinary conduct concerns, contact the maintainer through a GitHub issue
that does not expose private information. If a report contains personal data,
security details, threats, or another sensitive matter, use the private channel
described in [SECURITY.md](SECURITY.md).

Maintainers may edit or remove contributions, close discussions, temporarily
limit participation, or permanently exclude a participant when needed to
protect the community. Reports will be reviewed in context, shared only with
people needed to resolve them, and handled without retaliation.
