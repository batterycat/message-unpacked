# 學生模式與投影模式

### 學生自行練習

適合個人作答或小組討論。學生選擇答案後，會立即看到本題得分、理由、線索解析與安全行動；完成全部題目後會看到總分與逐題表現。

教師可以請學生先獨立作答，再兩人交換說明理由。

### 教師投影帶領

適合全班同步討論。學生選項會先停留在畫面上，教師按下「揭曉解析」後才會顯示結果、得分與案例影響。

建議流程：

1. 全班閱讀訊息。
2. 請不同意見的學生說明線索。
3. 教師統整後選擇班級答案。
4. 按「揭曉解析」。
5. 用案例影響與安全行動做收束。

投影模式不會收集每位學生的答案，也不會產生即時班級統計。

### 即時班級互動

這是另一個獨立入口，不會取代上面的純靜態投影模式。教師投影完整訊息並控制
開題與揭曉；學生用手機加入短期 room，手機只顯示題號與選項。

作答期間教師只看到加入與已作答人數。揭曉後才會顯示匿名選項分布、解析與
班級平均，不顯示誰選了什麼，也不建立個人成績或排名。完整步驟請見
「[即時班級互動模式](ban-ji-ji-shi-hu-dong.md)」。
