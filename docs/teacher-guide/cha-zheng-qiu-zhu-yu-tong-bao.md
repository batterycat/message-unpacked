# 查證、求助與通報

## 查證、求助與通報

遇到可疑訊息時，可以先使用這個順序：

1. **停下來**：不要點連結、回傳驗證碼或依訊息提供的電話操作。
2. **換入口查證**：從自己原本使用過的官方 App、網站或聯絡方式確認。
3. **找可信任的大人**：和家人、教師或其他可信任的人一起看。
4. **保留證據**：保留訊息、網址、帳號、轉帳紀錄與時間。

### 台灣在地資源

* [165 反詐騙諮詢專線](https://165.npa.gov.tw/)：不確定是否為詐騙時，可先諮詢與確認。
* [165 全民防騙網／打詐儀錶板](https://165.npa.gov.tw/)：查看官方防詐資訊與查詢入口。
* [網路詐騙通報查詢網](https://moda.gov.tw/ADI/antifraud/af-toolbox/16212)：查詢或通報網路詐騙相關資訊。
* **110**：正在發生、有人身安全疑慮，或需要警察協助時使用。

不要把「通報」當成每一題的固定答案。先判斷情境，再選擇合適的查證、求助或報案方式。
