# 歡迎｜Message, Unpacked.

## 歡迎來到教師手冊

這份手冊提供教師使用「Message, Unpacked.｜訊息拆解所」的課前準備、課堂帶領與課後討論建議。

如果第一次使用，請先閱讀「[課前快速開始](ke-qian-kuai-su-kai-shi.md)」；需要完整帶領流程時，再看「[課堂帶領腳本](ke-tang-dai-ling-jiao-ben.md)」。

想一起維護題庫時，請先讀「[新增題目與案例](xin-zeng-ti-mu-yu-an-li.md)」，從模板新增 YAML 題目，不需要修改程式碼。

這個工具的定位是：讓學生在安全、沒有真實損失的情境中，練習判讀簡訊、聊天訊息與 Email。

### 五分鐘快速開始

1. 開啟 [教師快速開課](https://batterycat.github.io/message-unpacked/zh-TW/teacher/)。
2. 選擇主題、活動時間與帶領方式。
3. 按下「產生活動連結」。
4. 投影活動網址或顯示 QR Code，讓學生加入。
5. 每題先討論，再查看解析與得分。

### 你可以選擇的模式

- **學生自行練習**：每位學生在自己的裝置上作答，完成後查看個人結果。
- **教師投影帶領**：全班看同一個畫面，由教師控制何時揭曉解析。
- **班級即時互動**：教師投影完整題目，學生手機只顯示題號與選項；需由網站
  另行設定短期 room 服務，詳見「[班級即時互動模式](ban-ji-ji-shi-hu-dong.md)」。

### 重要提醒

- 不需要登入，也不會要求學生輸入姓名。
- 純靜態模式不傳送答案；即時互動模式只送出隨機參與憑證與目前題目的選項，
  不建立個人歷程或排名。
- 題目中的連結是安全的示意內容，不會把學生帶到可疑網站。
- 真實案例的損失數字會標示資料來源與限定語；沒有公開資料時會明確標示未知。
- 網站不會替真實訊息做個案判斷，完整界線請見「[使用範圍、隱私與外部服務](shi-yong-fan-wei-yin-si-yu-wai-bu-fu-wu.md)」。
