# 歡迎｜Message, Unpacked.

## 歡迎來到教師手冊

這份手冊提供教師使用「Message, Unpacked.｜訊息拆解所」的課前準備、課堂帶領與課後討論建議。

這個工具的定位是：讓學生在安全、沒有真實損失的情境中，練習判讀簡訊、聊天訊息與 Email。

### 五分鐘快速開始

1. 開啟 [教師快速開課](https://batterycat.github.io/message-unpacked/zh-TW/teacher/)。
2. 選擇主題、活動時間與帶領方式。
3. 按下「產生活動連結」。
4. 投影活動網址或顯示 QR Code，讓學生加入。
5. 每題先討論，再查看解析與得分。

### 你可以選擇的模式

- **學生自行練習**：每位學生在自己的裝置上作答，完成後查看個人結果。
- **教師投影帶領**：全班看同一個畫面，由教師控制何時揭曉解析。

### 重要提醒

- 不需要登入，也不會要求學生輸入姓名。
- 題目中的連結是安全的示意內容，不會把學生帶到可疑網站。
- 真實案例的損失數字會標示資料來源與限定語；沒有公開資料時會明確標示未知。

程式碼與案例資料請見 [GitHub repository](https://github.com/batterycat/message-unpacked)。教師手冊與原創教學內容採 CC BY-SA 4.0。
