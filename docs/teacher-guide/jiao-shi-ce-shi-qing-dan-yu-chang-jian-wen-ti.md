# 教師測試清單與常見問題

## 教師測試清單與常見問題

### 第一次使用前

- [ ] 在教師自己的手機開啟教師頁。
- [ ] 選一個主題並產生活動連結。
- [ ] 點「開啟活動」確認題目數量與模式。
- [ ] 用另一支手機掃描 QR Code。
- [ ] 確認教室投影時文字大小與對比足夠。
- [ ] 先完成一題，確認解析、得分與案例資訊會出現。

### 如果連結無法開啟

1. 確認網址完整包含 `/activity/`。
2. 確認學校網路可以連到 `batterycat.github.io`。
3. 改用複製網址，不要只截取 QR Code 旁的部分文字。
4. 回到教師頁重新產生連結；案例下架或版本更新後，舊連結可能失效。

### 會不會留下學生資料？

不會。第一階段不要求登入、不輸入姓名，也沒有雲端作答紀錄。重新整理頁面後，當前活動進度可能會重設。

### 如何回報問題或提議新案例？

請到 [GitHub Issues](https://github.com/batterycat/message-unpacked/issues) 回報。案例提議請附上來源、適用年級、希望練習的手法，以及是否有敏感內容。
