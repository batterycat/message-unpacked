# 教師測試清單與常見問題

### 第一次使用前

- [ ] 在教師自己的手機開啟教師頁。
- [ ] 選一個主題並產生活動連結。
- [ ] 點「開啟活動」確認題目數量與模式。
- [ ] 用另一支手機掃描 QR Code。
- [ ] 確認教室投影時文字大小與對比足夠。
- [ ] 先完成一題，確認解析、得分與案例資訊會出現。
- [ ] 若要使用即時班級互動，用第二台裝置加入 room、送出答案、揭曉並結束
      room；也先準備純靜態投影模式作備援。

### 如果連結無法開啟

1. 確認網址完整包含 `/activity/`。
2. 確認學校網路可以連到展示站台。
3. 改用複製網址，不要只截取 QR Code 旁的部分文字。
4. 回到教師頁重新產生連結；案例下架或版本更新後，舊連結可能失效。

### 會不會留下學生資料？

純靜態模式不要求登入、不輸入姓名，也沒有雲端作答紀錄；重新整理後，當前
進度可能會重設。

即時班級互動同樣不輸入姓名或名單，但會把隨機參與憑證與目前題目的選項送到
短期 room。揭曉後刪除個別答案，只留下匿名統計，不建立個人歷程、排名或
成績。完整界線請見「[使用範圍、隱私與外部服務](shi-yong-fan-wei-yin-si-yu-wai-bu-fu-wu.md)」。

### 即時班級互動連不上怎麼辦？

1. 確認教師與學生裝置都能開啟展示網站。
2. 重新整理教師 room 與學生加入頁一次。
3. 若仍無法使用，回到教師頁改開純靜態「教師投影帶領」；靜態功能不依賴
   room 服務。
4. 公開 room 後端是體驗服務，可能因免費額度、容量或維護暫停，不應當作考試
   或必要校務系統。

### 如何回報問題或提議新案例？

請到 [GitHub Issues](https://github.com/batterycat/message-unpacked/issues) 回報。案例提議請附上來源、適用年級、希望練習的手法，以及是否有敏感內容。
