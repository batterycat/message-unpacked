# 課堂帶領腳本

以下是一個可在 20–30 分鐘內完成的流程，教師可以依班級狀況調整。

### 1. 開場（2–3 分鐘）

可以先問：

> 如果訊息看起來像是朋友、學校或熟悉的商店傳來的，你會先相信嗎？

說明今天的目標不是「猜中詐騙」，而是練習在做出動作前，找證據、確認來源、保護自己。

### 2. 每題流程（約 4–5 分鐘）

1. **靜默閱讀**：先讓學生自己讀完訊息。
2. **找線索**：請學生指出發訊者、要求、時間壓力、連結與語氣。
3. **先說理由，再選答案**：不要只問「真或假」，要追問「你看到了什麼？」
4. **揭曉解析**：說明答案、得分與更安全的下一步。
5. **連回生活**：請學生說出一個下次可以使用的查證方法。

### 3. 建議追問

- 這則訊息希望你立刻做什麼？
- 你能不能從原本使用過的入口重新確認？
- 如果不確定，可以找誰一起看？
- 哪些證據還不夠，不能只靠感覺下結論？

### 4. 課後收束

請學生完成一句話：

> 下次遇到類似訊息，我會先 \_\_\_\_\_\_，再 \_\_\_\_\_\_。

避免用「怎麼會上當」等責備語句。重點是建立可重複使用的安全習慣。
