# 敏感內容與課前提醒

多數案例可以直接帶領，以下幾則因為情境較敏感，建議你先自己看過一次再決定要不要用，
或準備好如何回應學生的反應。

**含「要求對大人保密」的案例**

- 《親戚手機壞掉的求助》（1–2 年級）
- 《遊戲裡的生日禮物》（1–2 年級）
- 《遊戲點數卡的刮刮碼》（1–2 年級）
- 《直播抽獎的獎品》（5–6 年級）
- 《按讚就能賺遊戲點數》（5–6 年級）

這幾則的訊息裡都有「不能告訴大人」這類要求。這是詐騙的常見手法，也是本課要教的
重點線索——但同樣一句話也是兒少誘拐（grooming）的核心話術。少數學生可能在詐騙以外
的情境遇過它。

帶領時的建議：

- 把重點放在「**要求你保密這件事本身就是警訊**」，這個原則同時適用於各種情況。
- 明確告訴學生：任何人叫你瞞著爸媽或老師，你都可以說出來，而且說出來不會有事。
- 如果有學生反應特別強烈、沉默或迴避，課後單獨關心，必要時聯繫輔導室。

**其他標記**

- `人身安全疑慮`：《朋友帳號傳來的緊急借款》（7–9 年級）涉及醫療急難情節。
- `恐懼訴求`：《轉發表情符號才不會倒楣》（3–4 年級）以厄運威脅製造焦慮，
  部分學生可能真的會感到不安。
- `兒少受詐`：《免費遊戲幣要用驗證碼換》（7–9 年級）改編自真實的國中生受害案件。
- `金錢損失`／`金融資料`：涉及金錢或金融資料的情境。若班上有學生家中曾遭遇詐騙，
  這類案例可能引發情緒反應。

**高中職的法律情境**

標記 `law` 的高中職案例會談到人頭帳戶、代收代轉、保留證據、報案或勞動程序。
這些內容是風險教育，不是對個別學生的法律判斷；請把重點放在「不要把帳戶、提款卡、
密碼交給別人使用」與「遇到疑慮要及早告訴家長、老師或 165」。若學生提到自己或家人
可能已經提供帳戶、付款或轉交資料，請課後私下關心，不要要求學生在全班說明細節，必要時
依學校流程轉介輔導或相關窗口。
