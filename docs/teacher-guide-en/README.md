# Welcome | Message, Unpacked.

This guide helps teachers prepare and facilitate **Message, Unpacked.**, an
open-source learning activity for practising how to assess texts, chats, and
emails before acting.

The current English release is a reviewed **grades 10–12, US-localized demo**.
It is not yet equivalent to the broader Traditional Chinese library. Read the
[pilot safety notice](sensitive-content.md) before first use: the English cases
do not yet provide per-case sensitive-content labels.

## Five-minute setup

1. Open [Quick class setup](https://batterycat.github.io/message-unpacked/en/teacher/).
2. Choose a **Static activity link** or **Live classroom interaction**.
3. Choose a topic and class length, then review every preselected case.
4. Create a share link for a static activity, or create a short-lived room for
   live responses.
5. Test the activity on a second device before sharing or projecting it.

## Three teaching modes

- **Student self-paced:** each learner answers and reviews explanations on
  their own device.
- **Teacher projector-led:** the class shares one screen and the teacher
  controls when the explanation appears.
- **Live classroom interaction:** the teacher projects the full case while
  student phones show only the case number and choices. This optional mode
  requires a compatible room service.

## Important boundaries

- No sign-in or student name is required.
- Static activities do not transmit answers. Live rooms use a random,
  short-lived participant credential and the current answer only.
- The site does not judge a learner's real message. Do not paste real messages,
  suspicious links, credentials, or personal data into project forms or
  classroom materials.
- Scores support discussion; they are not grades or measures of intelligence.
- The maintainer-hosted live service is for demonstration only. A school may
  deploy a compatible backend under its own privacy, security, and capacity
  policies.
